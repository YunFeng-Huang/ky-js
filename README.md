

# directive

> A Vue.js project

## Build Setup

``` 
npm install yf-vue-directive
import directive from 'yf-vue-directive'
Vue.use(directive)
#自定义指令

v-href = "{path:'index.html',query:{name:'123',age:'12'}}"

v-avatar="http://......"

v-copy="复制的值"

v-lazyload="img"  //img懒加载

v-scrollTop      //返回顶部

```
